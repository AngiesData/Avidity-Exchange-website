# Avidity Exchange

A clay-inspired, mobile-friendly static website for a flexible marketplace.

## Included

- Home page
- Digital Hub and four collection pages
- Marketplace and four collection pages
- Services and four collection pages
- Resources and four collection pages
- About, FAQ, Contact, Custom Request, Coming Soon, and Thank You pages
- Starter legal pages
- Netlify-ready contact forms
- Responsive navigation
- No framework or build step required

## Publish with GitHub and Netlify

1. Create a new GitHub repository named `avidity-exchange`.
2. Upload all files and folders from this project.
3. In Netlify, choose **Add new site → Import an existing project**.
4. Connect GitHub and select the repository.
5. Leave the build command blank.
6. Set the publish directory to `.`.
7. Deploy the site.
8. Test both forms.
9. Add your custom domain in Netlify before changing DNS records.
10. Keep the current paid site active until the new domain works with HTTPS.

## Before launch

- Replace placeholder text with your real business contact information.
- Review all legal templates with a qualified attorney.
- Add the payment provider only when products or services are ready.
- Never place paid digital files in a public GitHub repository.
- Add accurate disclosures for affiliate or third-party fulfilled products.
